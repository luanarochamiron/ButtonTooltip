# Share Button Tooltip with Social Media Icons

Um componente de interface de utilizador (UI) moderno, interativo e responsivo focado num sistema de compartilhamento estilizado através de um balão flutuante (Tooltip), desenvolvido com HTML5 e CSS3. O projeto apresenta microinterações suaves para exibição de redes sociais integradas a um botão de ação principal.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do botão de controle e da lista contendo os ícones de compartilhamento social.
* CSS3: Estilização completa, uso de bordas arredondadas (estilo pílula), sombras discretas e criação de formas geométricas.
* Flexbox: Alinhamento horizontal perfeitamente distribuído para os ícones e centralização absoluta de todo o bloco na viewport.

## Funcionalidades e Técnicas Aplicadas

* Tooltip Estilizado (Balão de Diálogo): O menu superior contendo os ícones apresenta formato elíptico alongado com um pequeno indicador triangular na base inferior, apontando diretamente para o botão de origem.
* Integração de Ícones Sociais: Espaço otimizado e simétrico que reúne ícones minimalistas das principais plataformas: Facebook, Twitter, Instagram e YouTube.
* Botão de Ação Arredondado: Botão principal "CANCEL" em formato de pílula colocado estrategicamente abaixo do tooltip, servindo como gatilho visual para fechar ou alternar o estado do menu.
* Design de Alto Contraste: Layout construído com componentes em branco puro que se destacam intensamente sobre um plano de fundo azul plano e sólido, garantindo excelente foco.
